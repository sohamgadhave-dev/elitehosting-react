## 📥 Installation & Setup

```bash
# Clone the repo
git clone https://github.com/sohamgadhave-dev/elitehosting-react.git
cd elitehosting-react

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
